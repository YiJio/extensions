🎉 Memokoi was updated to 2.0.0!

Thanks for continuing to use Memokoi.

Important changes

  • 🧠 Now built using React + TypeScript
  • 📂 Storage now saved to local instead of sync due to limits
    • This means sites only get saved to current device and will not sync across different devices for same browser profile.
  • 🏷️ Better tag creation via multiselect
  • 📥 JSON import/export with overwrite rules

What this means for existing users (you)

  • Unfortunately, how data is stored originally (sync storage) will not be used in the new update.
  • Sites will have to be added in once again.
    • Yes, this problem was exactly what Memokoi was designed to solve and yet the update has failed to accomplish that 😔.
  • However, your originally saved sites in sync storage is still ——hopefully, if you didn't manually delete them—— accessible by you and only you.
    • You can retrieve it by right-clicking the extension and clicking on "Inspect > Application > Extension storage > Sync" under "sites" key.
